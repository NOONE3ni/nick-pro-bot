# Nick Bot

WhatsApp bot with pairing code deployment and owner control.

## Commands

### Public
- `.ping` → Check bot online
- `.menu` → Show menu
- `.tts <text>` → Convert text to audio
- `.sticker` → Convert image to sticker

### Owner
- `.stats` → Show user count and uptime
- `.users` → List users
- `.block <number>` → Block a user
- `.unblock <number>` → Unblock a user
- `.restart` → Restart the bot

## Setup

1. Clone repo
2. `npm install`
3. Set `.env` variables
4. Deploy on Render or Node server# nick-pro-bot
